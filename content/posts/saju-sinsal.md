사주 명리학에서 **신살(神殺)**은 특별한 의미를 가진 글자들의 조합입니다. 길한 작용을 하는 '신(神)'과 흉한 작용을 하는 '살(殺)'로 구분됩니다. 신살을 모두 외울 필요는 없지만, 자주 사용되는 주요 신살은 알아두면 사주 해석에 도움이 됩니다.

## 길신(吉神) - 좋은 영향을 주는 신살

### 천을귀인(天乙貴人)

가장 대표적인 길신입니다. 어려울 때 귀인(도와주는 사람)이 나타나는 신살입니다.

**찾는 방법** (일간 기준)
| 일간 | 천을귀인 |
|------|---------|
| 甲·戊·庚 | 丑·未 |
| 乙·己 | 子·申 |
| 丙·丁 | 亥·酉 |
| 壬·癸 | 卯·巳 |
| 辛 | 午·寅 |

천을귀인이 사주에 있으면: 귀인의 도움을 받기 쉽고, 어려운 상황에서도 구해줄 사람이 나타납니다. 특히 년주나 월주에 천을귀인이 있으면 어릴 때부터 주변의 도움을 많이 받습니다.

### 문창귀인(文昌貴人)

학문과 총명함을 나타내는 신살입니다.

| 일간 | 문창귀인 |
|------|---------|
| 甲 | 巳 |
| 乙 | 午 |
| 丙 | 申 |
| 丁 | 酉 |
| 戊 | 申 |
| 己 | 酉 |
| 庚 | 亥 |
| 辛 | 子 |
| 壬 | 寅 |
| 癸 | 卯 |

문창귀인이 있으면 공부를 잘하고 문서·계약에 능합니다. 작가, 교육자, 연구자에게 유리한 신살입니다.

### 월덕귀인(月德貴人)

덕(德)을 상징하는 길신으로, 선량하고 복이 많은 사람을 나타냅니다.

- 寅·午·戌월생: 丙
- 申·子·辰월생: 壬
- 亥·卯·未월생: 甲
- 巳·酉·丑월생: 庚

## 역마살(驛馬殺)

이동, 여행, 변화를 상징하는 신살입니다.

| 생지(年支 또는 日支) | 역마 |
|---------------------|------|
| 寅·午·戌 | 申 |
| 申·子·辰 | 寅 |
| 巳·酉·丑 | 亥 |
| 亥·卯·未 | 巳 |

**역마살의 의미**
- 이동이 많고 여행을 자주 함
- 외국과의 인연이 있음
- 직업적으로 이동이 많은 일(영업, 무역, 운전, 출장)에 종사
- 부정적으로는 한 곳에 정착하기 어려움

역마가 길신과 함께하면 해외 발전, 무역 성공 등 좋은 방향으로 작용합니다.

## 화개살(華蓋殺)

예술적 감각과 고독을 나타내는 신살입니다.

| 생지 | 화개 |
|------|------|
| 寅·午·戌 | 戌 |
| 申·子·辰 | 辰 |
| 巳·酉·丑 | 丑 |
| 亥·卯·未 | 未 |

화개살이 있으면 예술적 감각이 뛰어나고 종교·철학에 관심이 많습니다. 고독을 즐기는 면이 있으며, 출세보다는 자신만의 세계를 추구합니다. 스님, 예술가, 철학자에게 많이 나타납니다.

## 백호대살(白虎大殺)

강한 에너지와 격렬함을 나타내는 신살입니다.

甲辰, 乙未, 丙戌, 丁丑, 戊辰, 己未, 庚戌, 辛丑, 壬辰, 癸未 일주에 해당합니다.

백호대살이 있으면 강한 에너지와 추진력이 있지만, 사고나 부상에 주의해야 합니다. 군인, 의사, 외과의, 운동선수 등 강인함이 필요한 직업에 많습니다.

## 양인살(羊刃殺)

강함과 폭력성을 나타내는 신살입니다.

| 일간 | 양인 |
|------|------|
| 甲 | 卯 |
| 丙·戊 | 午 |
| 庚 | 酉 |
| 壬 | 子 |

양인살이 있으면 의지력과 결단력이 강하지만 독단적일 수 있습니다. 잘 쓰이면 강한 리더십이 되고, 나쁘게 쓰이면 폭력성이 됩니다.

## 도화살(桃花殺)

이성 인기와 매력을 나타냅니다. (앞서 연애운 편에서 상세히 다룸)

## 신살 해석 시 주의사항

1. **신살은 참고사항**: 신살만으로 사주를 해석하면 안 됩니다. 사주 전체의 구조와 흐름 속에서 신살을 해석해야 합니다.

2. **길흉이 절대적이지 않음**: 같은 신살이라도 사주 구성에 따라 길하게 작용할 수도, 흉하게 작용할 수도 있습니다.

3. **현대적 해석 필요**: 역마살이 옛날에는 떠돌이를 의미했지만, 현대에는 해외 취업이나 글로벌 비즈니스로 해석할 수 있습니다.

4. **집착 금지**: 특정 신살에 지나치게 집착하거나 두려워하면 오히려 스스로 한계를 만들게 됩니다.

## 마치며

신살은 사주 해석을 풍부하게 해주는 도구입니다. 주요 신살을 알고 있으면 사주의 특성을 더 세밀하게 파악할 수 있습니다. 다만 신살은 보조적인 수단이며, 사주의 근본인 오행과 십성, 격국 해석이 우선입니다.
